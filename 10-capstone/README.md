# Course 10: Capstone Project

**Duration**: 9-10 Weeks  
**Status**: Not Started

## Course Description

Culminating project for the MScFE program. Students complete a comprehensive project applying all skills learned throughout the program.

## Capstone Options

- **MScFE 690**: Capstone Project (10 weeks)
- **MScFE 692**: Capstone Review (9 weeks)

## Learning Objectives

- Apply all program skills to a real-world financial engineering problem
- Demonstrate proficiency in computational, mathematical, and statistical methods
- Create professional technical reports
- Present findings effectively

## Project Structure

- Project proposal
- Literature review
- Methodology
- Implementation
- Results and analysis
- Final report and presentation
