# Course 1: Financial Markets

**Duration**: 7 Weeks  
**Status**: Not Started

## Course Description

Introduction to the world of professional finance: markets, products, participants, and regulation. Students explore activities within financial markets including trading, financing, brokering, pricing, hedging, optimizing, and managing risk.

## Learning Objectives

- Understand financial markets, products, and participants
- Identify significant factors affecting the financial industry
- Interact with web apps illustrating financial concepts
- Build foundation for mathematical and computational tools in financial engineering
